# Project Documentation Log

## Project Overview
This is a Next.js project that follows specific architectural patterns and best practices. The project emphasizes modern React patterns, TypeScript, and performance optimization. It's an interactive artist portfolio featuring animated artwork displays with different visual modes.

## Tech Stack
- **Framework**: Next.js with App Router
- **Language**: TypeScript
- **UI Components**: Shadcn UI
- **Styling**: Tailwind CSS, Custom CSS
- **Animation**: Framer Motion
- **State Management**: Server Components preferred, nuqs for URL state
- **Component Architecture**: Functional components with TypeScript interfaces

## Project Structure
```
/src
├── /app                    # Next.js App Router pages
│   ├── layout.tsx         # Root layout with Inter font
│   ├── page.tsx           # Home page with dynamic ClientWrapper
│   └── globals.css        # Global styles
├── /components            # Shared components
│   ├── ClientWrapper.tsx  # Client-side wrapper with Suspense
│   ├── EnhancedPortfolio2.tsx # Main portfolio component
│   └── ArtGallery.jsx    # Alternative gallery implementation
└── /styles
    └── hover-effects.css  # Custom hover animations
```

```typescript

export default function ClientWrapper() {
  return (
    <Suspense fallback={

  );
}
```
